# Introdução

Esta é uma landing page de um Resort.

Este projeto foi desenvolvido para solidificar conhecimentos sobre ReactJS.

Para vê-lo em ação, acesse este link: https://summer-hotel-landing-page.vercel.app/

## Requisitos
Para instalação e utilização do projeto em sua máquina é necessário ter instalado NodeJs.

## Instalação

Use o github para clonar o projeto:

```bash
git clone
```
Com isso, use o comando

```bash
npm install create-react-app
```

Por fim, basta executá-lo com

```bash
npm start
```
